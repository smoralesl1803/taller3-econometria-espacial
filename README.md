# Taller 3 - Econometría Espacial

Este repositorio contiene el desarrollo del **Taller 3 de Econometría Espacial**, centrado en el análisis de la **concentración, especialización y aglomeración espacial de la industria del calzado en Cali en 2005**.

## Contenido del repositorio

- `Taller3.qmd`: archivo fuente en Quarto con el código y la redacción del taller.
- `docs/index.html`: versión renderizada del taller para visualización como página web.
- `.nojekyll`: archivo auxiliar para la publicación del sitio con GitHub Pages.

## Temas analizados

El trabajo incluye:

1. **Mapa de densidad y distribución de puntos** de las empresas del sector calzado.
2. **Índices de concentración espacial**, incluyendo:
   - Índice de Herfindahl-Hirschman (IHH)
   - Índice de Ellison y Glaeser (EG)
   - Índice de Maurel y Sédillot (MS)
   - Location Quotient (LQ)
   - Industry Quotient (IQ)
3. **Mapa del LQ** para identificar especialización territorial.
4. **Funciones de distancia**:
   - Función \(K_d\) de Duranton y Overman (2005)
   - Función \(M\) de Marcon y Puech (2010)

## Objetivo

Analizar la localización de la industria del calzado en Cali en 2005, identificando patrones de concentración geográfica, especialización territorial y distancias características de aglomeración, con una interpretación económica basada en el contexto urbano e industrial de la ciudad.
